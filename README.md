## [FeatherBlaze's Blog](https://github.com/fblaze62/blog) Source Code

[Preview](https://blog.fblaze62.top/?_blank)

1. Clone the project

```bash
git clone https://github.com/fblaze62/blog-source.git
```

2. Install dependencies

```bash
npm install
```

3. Build

```bash
npx hexo clean
npx hexo generate
npx hexo deploy
```

4. Deploy

   Using GitHub Pages or Cloudflare Pages.

5. Configuration

   Refer to the [Hexo documentation](https://hexo.io/docs/?_blank) and [Volantis theme documentation](https://volantis.js.org/v7/getting-started/?_blank).
